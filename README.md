This is a simple Tic-Tac-Toe game built using Python and Tkinter, designed to resemble a phone screen. Players take turns selecting empty cells to mark **X** or **O**, with the goal of forming a row, column, or diagonal of three matching symbols to win. If all cells are filled without a winner, the game ends in a draw. The game features automatic win detection and resets after a match.

## Features
- Automatic win detection.
- Game resets after each match.
- Playable with **X** or **O**.
- Designed to resemble a phone screen for the user interface.
